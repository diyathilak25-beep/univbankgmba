# Universal Bank Personal Loan Analysis Dashboard

A comprehensive Streamlit dashboard for analyzing customer data and predicting personal loan acceptance.

## 🎯 Objective

To understand which customers are more likely to accept a Personal Loan offer.

## 📊 Features

- **Descriptive Analytics**: Customer demographics and loan acceptance patterns
- **Diagnostic Analytics**: Key drivers and correlation analysis
- **Predictive Analytics**: ML models for loan acceptance prediction
- **Prescriptive Analytics**: Marketing recommendations
- **Personalized Offers**: Automated offer generation

## 🚀 Quick Start

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Add `UniversalBank.csv` to root folder
4. Run: `streamlit run app.py`
